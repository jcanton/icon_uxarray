## Install it from PyPI

```bash
pip install icon_uxarray
```

## Usage

Convert an icon grid to be UGRID-compatible

```py
from icon_uxarray import iconGrid2Ugrid
new_file_name = iconGrid2Ugrid(iconGrid_fname='some_icon_grid.nc')

```
